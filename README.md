# Virtual Playground

A PC Emulator implemented by WebAssembly.

- [preview](https://nerry.jp/vpc/)

## Implemented hardware

- CPU: 80186?
  - Some features are missing
- Memory: 640KB ought to be enough for anybody.
- I/O:
  - i8259 PIC
  - i8254 Timer & Sound
  - UART
  - RTC
  - FDC (non standard interface)
